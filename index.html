A simple package manager,
written in bash,
and made for bash.

Install bashpack :

Simply past this line in your MacOs terminal or linux shell :

  $  /bin/bash -c "$(curl -fsSL https://bashpack.me/pckg/install.sh)"

What does bashpack do ?

Bashpack is an open-source package manager allowing you to download, update, and host your projects and open-source script written in bash.

How it works ?

bashpack [args] [Pckg Name]

 bashpack -i $pckg-name # Install a pckg
 bashpack -d $pckg-name # Delete a pckg
 bashpack -u $pckg-name # Check/Update
 bashpack -s $pckg-name # Get info
 bashpack -h            # Show Help
